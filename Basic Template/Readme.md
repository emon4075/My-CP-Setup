# Basic Template (C++)

## Explanation

### Purpose
This template is designed for competitive programming with a focus on setting up fast I/O operations.

### Key Features
- **Essential Headers:** Includes commonly used C++ headers.
- **Common Macros:** Defines frequently used macros such as `endl` for line breaks and `ll` for long long integer type.
- **Fast I/O Setup:** Incorporates a function to optimize input and output operations, reducing execution time during competitions.
